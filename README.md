# SV Daily Bot — Сила Ветра

Ежедневный отчёт по рекламе в Telegram: расходы Директ + доход/конверсии Метрики.

## Деплой на Railway

1. Зайди на railway.app → New Project → Deploy from GitHub
2. Загрузи эти файлы в репозиторий GitHub
3. Railway автоматически найдёт Procfile и запустит бота

## Файлы
- bot.py — основной код
- requirements.txt — зависимости
- Procfile — команда запуска для Railway
